🚀 3.0 核心进化：工业化与理性穿透
在 V3.0.5 中，我们完成了从“模拟逻辑”到“物理执行”的跨越：

DeepSeek R1 Reasoner 集成：原生支持 R1 思维链（Reasoning Chain），在输出审计报告前，系统会实时展示其“逻辑推演过程”，确保审计深度直达物理基底。

LTPE 物理审计协议（全量实装）：

[L] 空间 (Location/Space)：物理方位、位移矢量、重力/流体力学约束（如：头骨为何定向移动？）。

[T] 时间 (Time)：热力学第二定律的应用，锁定转场窗口，通过时间线熵增识别逻辑断层。

[P] 物理性质 (Physicality)：物质残留、化学腐蚀（铜绿）、工具切迹、机械强度。

[E] 暗能量 (Energy/Social Capital)：利益流向、暗处的社交币补偿、隐形成本白嫖。

🛠️ 运行流程 (The Why-Skill Workflow)
卷宗录入：支持多行长文本粘贴，通过 :send 指令触发高强度扫描。

物理降噪 (Extractor)：本地 Python 脚本自动剔除感性修辞、图片标签、时间戳噪音。

阶段一：情报探针 (Stage 1)：AI 模拟三个 Agent（物理猎犬、系统黑客、魔鬼代言人）交叉扫描，输出《核心情报缺口清单》。

阶段二：终极审计 (Stage 2)：基于物理回填数据，生成带有 异常指数 (0-10) 的冷酷报告。

🔍 3.0 特色模块
流式输出 (Streaming)：实时打字机效果，同步抓取 DeepSeek 思考内容，让逻辑推演过程透明化。

物理穿透代理：内置强行代理注入模块（默认 127.0.0.1:7897），确保在复杂网络环境下依然具备极高响应速度。

自动质检官 (Sanitizer)：审计报告完成后，本地脚本自动进行语义质检，强制剔除“可能、大概”等模糊词汇。
