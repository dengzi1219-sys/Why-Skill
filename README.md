## 🚀 核心价值
本项目是一套基于“奥卡姆剃刀”与“芒格逆向思维”的 AI 审计协议。它能帮你：
1. **剥离烟雾弹**：看穿对方用来掩盖私心的“大道理”和“高尚借口”。
2. **追踪物理变量**：先查行程坐标，后谈精神动机。
3. **情报回填**：不仅告诉你对方在撒谎，还给你一份**【调查指南】**，告诉你该去问谁、怎么问。

## 🛠️ 如何使用
1. 复制本项目中的 `WHY_ALGORITHM.md` 全文。
2. 粘贴给具备思考能力的 AI（推荐 DeepSeek-V4-Pro 或 Claude 3.5）。
3. 输入你的“冲突案例”，AI 将自动出具审计报告。

## 🔍 特色模块：情报探针 (Intel Inquiry)
针对你“不知道该问什么”的社交盲区，本引擎提供：
- **侧向探针**：不打草惊蛇，通过询问“细节的细节”让谎言由于无法闭环而自燃。
- **第三方穿透**：教你如何低姿态地从共同好友处调取“后台日志”。
